# Hero Fotoğrafı

Bu klasör ana sayfadaki büyük fotoğrafı içerir.

## Kullanım

- **1.jpeg**: Ana sayfa hero bölümünde gösterilen fotoğraf
- Fotoğraf boyutu: En az 800x1000 piksel önerilir (dikey format)
- Format: JPEG

## Fotoğraf Değiştirme

1. Yeni fotoğrafınızı bu klasöre `1.jpeg` olarak kaydedin
2. Eski fotoğrafı silin veya üzerine yazın
3. Değişiklik otomatik olarak sitede görünecektir

## Öneriler

- Dikey (portrait) format daha iyi görünür
- Yüksek çözünürlük kullanın
- Dosya boyutu 1MB'ın altında olmalı (hız için)
